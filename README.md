# Personal Book Manager 📚

Hey there! This is my Personal Book Manager app — a simple web application to keep track of your book collection. You can add new books, see all your books, update details, and remove books you’re done with. It’s responsive, fast, and easy to use.

## Features ✨

- Add books with title, author, status, and tags
- See all your books in a neat list
- Update any book’s info
- Delete books you no longer need
- Filter the books by tags and status

## Tech Stack 🛠️

- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MongoDB Atlas
- **Styling:** Vanilla css (module.css)
- **API:** RESTful

---

## Getting Started 🚀

These instructions will get the project up and running on your local machine.

### 1. Clone the repo

git clone https://github.com/AbidDhankwala123/Personal-Book-Manager.git
cd Personal-Book-Manager

2. Install dependencies

Backend:

cd server
npm install
npm i express mongoose dotenv cors bcrypt jsonwebtoken

Frontend:

cd ../client
npm install
npm i axios react-router-dom react-toastify

3. Run the app

Start the backend server:

cd server
npm start

It should run at http://localhost:5000.

Start the frontend:

cd client
npm start

It should open in your browser at http://localhost:3000.
